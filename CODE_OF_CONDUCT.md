# Code of conduct

Be respectful, assume good faith, argue about code and costs rather than
people. Harassment, discrimination and personal attacks are not tolerated in
issues, PRs or any project space. Reports go to the maintainer via GitHub;
consequences range from a warning to a ban at the maintainer's discretion.

This is a condensed statement in the spirit of the
[Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/);
when in doubt, its standards apply.
